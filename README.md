# Zeelink — แพลตฟอร์มพอร์ตโฟลิโอเชิงสังคมสำหรับคนไทย

แพลตฟอร์มที่เชื่อมครีเอเตอร์และคนไทยเข้าด้วยกัน คล้าย Linktree แต่มีของเพิ่ม:
แผนที่แสดงคนใกล้เคียง · ระบบโหวตคำถาม · ปรับธีมสีหน้าเว็บได้ · แผงแอดมิน · และส่วน AI

## หน้าต่าง ๆ ในแอป
- **หน้าแรก (Landing)** — หน้าโปรโมท พร้อม Pixel Mascot (ตัวการ์ตูนพิกเซลที่ขับตามเมาส์) + อนิเมชั่นเลื่อนปรากฏ
- **ฟีด (Feed)** 🆕 — เหมือนฟีด Facebook: โพสต์ข้อความ / รูป / วิดีโอ ถูกใจ คอมเมนต์ แชร์
- **แดชบอร์ด (Dashboard)** — พอร์ตโฟลิโอ: สร้างลิงก์แพลตฟอร์ม (แบบ BIO) ได้จริง + แผนที่ Leaflet แชร์ตำแหน่งเรียลไทม์ + เลือกที่อยู่ครบ 77 จังหวัด
- **สำรวจ (Explore)** — แผนที่แสดงโปรไฟล์คนใกล้เคียง (ครอบคลุม 77 จังหวัด)
- **โหวต (Vote)** — ถาม/โหวตคำถามกับชุมชน
- **โปรไฟล์ (/username)** — หน้าโปรไฟล์ส่วนตัวรวมลิงก์ (ปรับธีมสีได้)
- **แอดมิน (/admin)** — จัดการผู้ใช้ / ป็อปอัปประกาศ

## เทคโนโลยี
- หน้าบ้าน: React 19 + Vite + TypeScript
- หลังบ้าน: Supabase (ฐานข้อมูล + ไฟล์)
- แผนที่: Leaflet · ไอคอน: lucide-react
- AI: Gemini (สคริปต์อยู่ที่ `Server/ai.js`)

## รันดูบนเครื่องตัวเอง (โหมด demo)
```bash
npm install      # ติดตั้งครั้งแรก
npm run dev      # เปิดเซิร์ฟเวอร์ → http://localhost:3000
```
โหมด demo ดูหน้าเว็บได้เลยโดยไม่ต้องต่อฐานข้อมูล (มีข้อมูลจำลอง)
เข้าหน้าแอดมิน demo ได้โดยล็อกอินด้วยอีเมล `zbcd1053@gmail.com` และรหัสอะไรก็ได้

## ต่อฐานข้อมูลให้ระบบสมาชิกทำงานจริง
1. สร้างโปรเจกต์ฟรีที่ https://supabase.com
2. เปิด **SQL Editor** → รันไฟล์ `supabase/schema.sql` (สร้างตารางพื้นฐาน)
3. **รันเพิ่ม** ไฟล์ `supabase/migration-feed-latlng.sql` เพื่อเพิ่มตาราง `posts`/`post_comments` และคอลัมน์ `lat/lng` (รันซ้ำได้ ไม่ error)
4. แก้ไฟล์ `.env` (คัดลอกจาก `.env.example`) ใส่ Project URL + anon key จาก Supabase Settings → API
5. รีสตาร์ท `npm run dev` → สมัครสมาชิก / ล็อกอิน / โปรไฟล์ / ฟีด จะทำงานจริง

> 💡 ถ้ายังไม่มี `.env` แอปจะเข้า **โหมด Demo** โดยอัตโนมัติ (มีข้อมูลจำลอง ทุกหน้าใช้งานได้ปกติ) ไม่พังหน้าจอ
> 🤖 แอดมินจำลอง (AiMascot) ใช้ Groq API (ตั้งค่าใน `.env` → `VITE_AI_API_KEY`) หากไม่ตั้งค่าจะตกไปใช้คำตอบสำรองภาษาไทย

## ฟีเจอร์ใหม่ (อัปเดตล่าสุด)
- **Pixel Mascot** — ตัวการ์ตูนพิกเซลขับตามเมาส์ คลิกเพื่อให้ทักทาย (หน้าแรก + ฟีด)
- **หน้าแรกมีชีวิตชีวา** — อนิเมชั่นเลื่อนปรากฏ (scroll-reveal) + องค์ประกอบลอยตัว
- **ฟีดแบบ Facebook** — โพสต์/รูป/วิดีโอ ถูกใจ คอมเมนต์ แชร์
- **แดชบอร์ดพอร์ตโฟลิโอ** — สร้างลิงก์แพลตฟอร์ม (BIO-style) + แผนที่แชร์ตำแหน่งเรียลไทม์ + เลือกจังหวัดครบ 77 จังหวัด
- **ที่อยู่ครบ 77 จังหวัด** — อัปเดต `constants.ts` เรียงตามภูมิภาค (กลาง/อีสาน/เหนือ/ใต้)
